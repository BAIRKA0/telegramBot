Для работы с ботом требуется изменить id владельца, потому что бот принимает команды только от владельца, 
но у него так же есть 2 команды которые может использовать любой пользователь (/help и /run)

Для изменения id владельца, нужно открыть файл config.ini в блокноте и изменить цифры в "owner_id =",
либо написать команду /change_owner боту в телеграмме(но после этого нужно перезапустить программу)

Все команды: 
/change - изменение файла номеров (вместе с командой надо отправлять excel файл) 
/invite - приглашение пользователей в определенный канал
/run - приглашение пользователей в канал, в котором написали команду
/add - добавление номера в excel (пример: /add 7xxxxxxxxxx)
/del - Удаление номера из excel (пример: /del 7xxxxxxxxxx)
/change_owner - меняет владельца бота и в конце нужно указать пароль (пример /change_owner 7xxxxxxxxxx 1234)

Различие между run и invite
Чтобы использовать команду /run нужно добавить бота в канал с сделать его админом, далее в канале нужно отправить 
данную команду (если написать эту команду ему в личные сообщения, то он ответит 'Эту команду нужно писать в канале').

Команда /invite пишется в личные сообщения и только от владельца
